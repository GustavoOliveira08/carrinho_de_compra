# 🛒 Sistema de Cadastro de Produtos em Java

# 📌 Objetivo do Código

O objetivo do programa é criar um sistema simples de cadastro de produtos utilizando duas listas separadas: uma para os nomes e outra para os preços.

O sistema permite cadastrar vários produtos até que o usuário digite `"fim"`. Após isso, o programa exibe os produtos cadastrados, calcula o valor total e identifica o produto mais caro.

---

# 📚 Estruturas de Dados

Foram utilizadas duas listas para separar os dados dos produtos e seus preços, mantendo as informações organizadas através do índice de cada item.

---

# 🔢 Variáveis Utilizadas

As variáveis foram utilizadas para:

- armazenar os dados digitados;
- controlar o encerramento do programa;
- calcular o valor total;
- identificar o produto mais caro.

---

# 🔁 Estrutura de Repetição

O `while(true)` foi utilizado para manter o sistema funcionando continuamente, permitindo múltiplos cadastros sem definir uma quantidade fixa de produtos.
A palavra `"fim"` foi utilizada como condição de parada para que o usuário pudesse encerrar o cadastro quando desejasse.
A comparação ignorando letras maiúsculas e minúsculas foi utilizada para tornar o programa mais flexível durante a digitação.
Os produtos e preços são armazenados separadamente para facilitar a organização e exibição dos dados.


# ✅ Conceitos Utilizados

| Conceito | Objetivo |
|---|---|
| Listas | Armazenar os dados |
| Repetição | Permitir múltiplos cadastros |
| Condições | Controlar decisões do programa |
| Comparações | Encontrar o maior valor |
| Acumuladores | Somar os preços |

---

# 🎯 Conclusão

O programa atende o objetivo proposto pelo exercício, permitindo cadastrar produtos dinamicamente, calcular o valor total e identificar o produto mais caro utilizando listas, repetições e comparações.
